# Aim of blpapi jar:

The blpapi jar is used by Eclipse Tradista in order to connect to Bloomberg. 
This allows retrieving Market Data, in real time or batch mode.

# Location

The jar comes from the Bloomberg website, from this download page: https://www.bloomberg.com/professional/support/api-library/
